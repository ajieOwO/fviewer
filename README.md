## 项目简介

一个没什么特点的文件浏览命令行工具

## 镜像仓库说明

本项目维护以下两个镜像仓库

- [Gitee镜像仓库](https://gitee.com/ajieOwO/fviewer)仅同步，不处理贡献和issues
- [GitHub镜像仓库](https://github.com/ajieOwO/fviewer)主镜像仓库，处理贡献和issues

## 编译指南

```bash
cargo run
```

## 使用说明

```bash
fviewer -h
```
